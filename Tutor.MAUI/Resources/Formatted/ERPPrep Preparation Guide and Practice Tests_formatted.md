## SAP Certification Exams

- **Associate-** and **professional-level certifications** require a single exam.  
- **Specialty certifications** require candidates to pass both the specialty and associate-level exam.  
- Exam vouchers can be purchased through the **SAP website**, and exams are taken at a **SAP Training Center** or through **Pearson VUE**.  
- Some certifications are available through the **SAP Certification in the Cloud** program. For an annual subscription of **$550**, candidates may take up to six exams in a 12-month period.  
- The **Certification in the Cloud** fee is roughly equivalent to the cost of one exam at a physical testing center.  
- For those seeking multiple certifications, reviewing available cloud certs is recommended.

Most certifications are at the **associate level**, but many **professional** and **specialist certifications** are also available.

## SAP NetWeaver

**SAP NetWeaver** certifications focus on administering SAP applications such as **ERP, CRM, Java, ABAP**, and technologies like **business warehouse, portal, integration technologies**, and **development**.

Examples of current certifications:

- **Technology Consultant OS/DB Migration for SAP NetWeaver 7.4**
- **Business Warehouse Application Associate - BI with SAP NetWeaver 7.0**
- **Portal Technology Associate - SAP Enterprise Portal 7.4.0**
- **Process Integration Technologies Development Associate/Technology Associate - SAP NetWeaver 7.31**
- **Programming ABAP Development Associate - ABAP with SAP NetWeaver 7.40 or 7.50**

Numerous **NetWeaver training opportunities** including project team training are available.

## SAP Solution Manager

**SAP Solution Manager** is key for managing complete SAP-based solution environments, including **selection**, **implementation**, **daily operations**, and **system reliability**.

Training is classified into subcategories:

- General solution overviews  
- Project management  
- End-to-end solutions  
- How to configure and use

SAP offers certifications such as **Certified Technology Associate** credentials focused on:

- Change control management  
- Business process monitoring  
- Business process interface monitoring  

Each certification includes a recommended training roadmap.

An important certification is the **SAP Certified Associate Project Manager**, which validates skills specific to **managing SAP projects**. Exam topics include:

- **Data management**  
- **Key deliverables in work stream**  
- **Business process management**  
- **Managing technical solutions including ASAP methodology**  
- **Implementing SAP projects**  
- **Enabling tools**  
- **Key ASAP concepts**  
- **Training within the SAP methodology**  
- **Change management**  
- **SAP NetWeaver capabilities**  
- **Test and cutover management**  
- Running SAP “like a factory” with enabling tools

## SAP Sybase Adaptive Server Enterprise (SAP Sybase ASE)

SAP offers a full database and technologies training and certification portfolio covering:

- **DB administration**  
- **Integration**  
- **Security**  
- **Development**  
- **Performance**

Other concentration areas include **SAP IQ, SAP Replication Server, SAP Power Builder**, and **SAP Power Designer**.

The **SAP ASE portfolio** has two subcategories:

- **ASE Administration**  
- **ASE Development**

### ASE Administration

- Certification: **Technology Associate – Database Administrator for SAP Sybase ASE 15.7**  
- Exam topics:  
  - Database backup  
  - Server configuration  
  - Tuning with Logical Process Management  
  - Transactions using Transact-SQL functions  
  - Architecture and installation  
  - Maintenance  
  - Security  
  - Table management  
  - Database administration  
  - Data analysis  

Training includes **SAP Adaptive Server Enterprise fast-track course** and other courses such as:

- System and Database Administration  
- Performance and Tuning: Configuring Adaptive Server Enterprise  
- Powering Up with Sybase Adaptive Server

### ASE Development

- Certification: **SAP Certified Technology Associate – Development Consultant for SAP Sybase ASE 15.7**  
- Exam topics:  
  - Data analysis  
  - Query optimization and statistics  
  - Query access methods  
  - Join processing  
  - Table and index structure  
  - Transaction concepts  
  - Stored procedures and optimization  
  - ASE performance tuning and basics  

Recommended training:

- Fast Track to SAP Adaptive Server Enterprise  
- Performance and Tuning: Adaptive Server Enterprise 16.0 Query Optimization

Two main levels exist:

- **Associate certification**: For people with no or little SAP experience
